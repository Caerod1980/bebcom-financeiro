# 🍺 Bebcom Financeiro

Sistema de gestão financeira completo para Bebidas & Companhia

## 🚀 Tecnologias

- **Backend**: Node.js + Express + MongoDB
- **Frontend**: React + TailwindCSS + Vite
- **Infra**: GitHub + Azure + MongoDB Atlas

## 📦 Instalação

### Backend

```bash
cd backend
npm install
cp .env.example .env
# Edite .env com suas credenciais do MongoDB
npm run seed  # Cria admin e dados iniciais
npm run dev
